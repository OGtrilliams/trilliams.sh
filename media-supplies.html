DIY Media Server - Hardware requirements:

A PC with the following:

- Windows (10 or greater), MacOS (10.13 or higher), or Linux OS

- 4GB or more RAM

- 4 CPUs or greater

- at least 40GB disk

HDMI cable

Some type of monitor (same as above)

Keyboard & mouse

Optional:

external harddrive for storing digital media
