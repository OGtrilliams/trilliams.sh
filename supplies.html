DIY DNS Server hardware requirements:

Raspberry Pi

Minimum specs: RasPi Zero 2 - 512MB RAM

Best option: RasPi 4 - 2GB RAM (anything more is overkill)

MicroSD card(doesn't have to be this one, but the cardked comes preinstalled with PiOS)

HDMI Cable

Some type of monitor - can be a computer monitor, TV, etc...

USB cable

Keyboard & mouse

Optional:

Ethernet cable

RasPi case

Heatsink kit

DIY Media Server - hardware requirements:

A PC with the following:

- Windows (10 or greater), MacOS (10.13 or higher)

- 4GB or more RAM

- 4 CPUs or greater

- at least 40GB disk

Optional: external harddrive for storing digital media

HDMI cable

Some type of monitor (same as above)

Keyboard & mouse
